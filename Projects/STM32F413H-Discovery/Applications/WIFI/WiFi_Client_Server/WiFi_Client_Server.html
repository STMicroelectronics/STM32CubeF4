Client Server using ES_WiFi module

Copyright 2017 STMicroelectronics

 


This application shows how to use the ES_WiFi module to perform a TCP Client mode using STM32 Cube HAL.
It demonstrates how to set up a client program, connect it to a TCP server.

There are several TCP server applications available.
TCP server application for Android is one that can be setup as a TCP/IP Server.
1. Connect the Android into the same WIFI network as the TCP client.
2. Get the Android IP address. It should be set in main.c of the TCP client.
3. Set the same port number as set in TCP Client (port 8002).
4. Press Create TCP server.
5. Execute TCP client application. The data will be sent to the TCP server.


  • The Es_wifi module and the mobile phone start to connect to the same WiFi access point configuration.
  • The mobile phone is running as a TCP server while the Es_Wifi is initiated as a Client. 
  • While connection by the client is still alive, the client and server can communicate. The client sends a message "STM32: Hello".
