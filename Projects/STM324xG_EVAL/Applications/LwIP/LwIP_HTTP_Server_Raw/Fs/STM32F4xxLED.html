STM32F4xx Leds control          

Home page Led control ADC status bar


This page allows you to control the four LEDs: LED1, LED2, LED3 and LED4 located in the STM324xG-EVAL board. To put on/off
a LED you have to check/uncheck its corresponding checkbox. Then you have to click on "Send" button to submit the new LEDs
configuration. Finally check in the STM324xG-EVAL board that you get the desired LEDs are putted on/off.
